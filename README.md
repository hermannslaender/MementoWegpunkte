# Wegpunkte

## MementoWegpunkt
Holt sich über API zwei meiner Memento Datenbanken und generiert daraus eine GPX-Datei für mein Navi und eine JSON-Datei für die Karte des GC-Wizard.

## GarminWegpunkte
Kann eine GPX von meinem Garmin in eine Tabelle einlesen. Dort kann ich löschen und auf der Karte anzeigen. Export zu Memento fehlt noch.
